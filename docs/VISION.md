# Network City Vision

## Mission

Make computer networking as intuitive to understand as watching traffic move through a city.

Network City is a visual Cisco networking simulator and learning game. Players design, configure, troubleshoot, and expand the network infrastructure supporting a growing city.

The project combines:

- authentic Cisco-style CLI configuration
- accurate network simulation
- visual explanations of invisible network behavior
- scenario-based CCNA learning
- city-building progression
- adaptive tutoring

## Core Experience

The player observes a city, identifies a connectivity or infrastructure problem, configures network devices through a Cisco-style CLI, and watches the effects of those commands become visible in the city.

The primary learning loop is:

1. Observe the city.
2. Identify the networking problem.
3. Form a prediction.
4. Configure the network.
5. Watch the network respond.
6. Verify the result with Cisco commands.
7. Explain why the solution worked.
8. Expand the city and encounter a more complex problem.

## Product Promise

Every Cisco command should have a visible consequence.

Every networking concept should solve a problem the player has already experienced.

The visuals must explain real networking behavior without replacing the technical details.

## Initial Scope

The first playable version will include:

- one small city
- two LANs
- two routers
- two switches
- two endpoint buildings
- IPv4 addressing
- router interfaces
- static routing
- ping
- packet visualization
- `show ip interface brief`
- `show ip route`
- route visualization

## Long-Term Vision
