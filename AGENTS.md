# AGENTS.md

## Project

Network City

## Mission

Build the world's best visual networking simulator.

The project teaches Cisco networking through an overhead city simulation.

## Principles

- The World is the source of truth.
- The simulation engine never depends on React.
- The simulation engine never depends on Pixi.
- Every Cisco command changes simulation state.
- Rendering only visualizes simulation state.
- Prefer composition over inheritance.
- Keep commits small.
- Explain architectural decisions.
- Never add abstractions before they are needed.
